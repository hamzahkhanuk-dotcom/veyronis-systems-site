# VEYRONIS Systems — Premium Robot Storefront

A 6-page React + Tailwind + Framer Motion premium storefront concept for VEYRONIS Systems.

## Pages
- Home
- VEYRONIS-01 Product
- Engineering
- Operator AI
- Private Access / Pricing
- Company / Contact

## Run locally
```bash
npm install
npm run dev
```

Open the local URL shown in Terminal, usually `http://localhost:5173`.

## Build for production
```bash
npm run build
npm run preview
```

## Notes
- Contact email currently used: `access@veyronissystems.com`
- Extra team members are fictional demo placeholders.
- Pricing is positioned as coming-soon private access / concept reservation pricing.
